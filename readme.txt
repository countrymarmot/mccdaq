# Original Readme

This is the README file for MCCDAQ

MCCDAQ USB1208LS is an interface USB-DAQ Data Acquisition board running only on windows (XP pro,W2K,Me,W98)

Installation
- connect USB1208LS
- install MCCDAQ driver (see CDROM libraries)
- install MCCDAQ python package
